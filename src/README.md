# Sanitized Reference Code

This directory contains a public-safe miniature pipeline that mirrors the trust boundary of the private project:

1. user question
2. mock schema context
3. prompt preview
4. semantic plan
5. server-side SQL rendering
6. read-only validation
7. mock execution result
8. trace output for human review

It intentionally does not include production controllers, prompts, database adapters, uploads, evaluation corpora, or internal schema names.
